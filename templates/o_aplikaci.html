O aplikaci

Cíl projektu

Cílem této jednoduché webové knihovny je naučit se pracovat s frameworkem Flask a vytvořit dynamickou webovou aplikaci, která zobrazuje seznam knih.

Použité technologie

  • Python 3.10+
  • Flask (webový mikroframework)
  • HTML + CSS

Funkce aplikace

  • Zobrazení seznamu knih
  • Rozlišení mezi dostupnými a vypůjčenými knihami
  • Šablonovací systém s děděním a include
  • Navigační menu
